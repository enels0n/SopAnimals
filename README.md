# SopAnimals

SopAnimals is a Bukkit/Spigot plugin that expands animal taming and breeding with ownership, hunger, reproduction timers, commands, and configurable mob-specific behavior.

## Features

- Custom tame and feed item lists per mob
- Ownership transfer
- Pregnancy, child growth, and lifetime timers
- Reward and automation commands from config
- PlaceholderAPI-aware level checks
- Shared text handling through SopLib

## Requirements

- Java 8+
- Spigot/Paper 1.20.4
- SopLib
- PlaceholderAPI

## Commands

- `/sopanimals`
- `/sopanimals reload`

## Notes

- SopLib is required as a separate plugin dependency.
- The plugin keeps old `totalanimals` compatibility as an alias.
